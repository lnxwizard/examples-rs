# examples.rs
Code Examples for Beginner Rust Developers.

# Running Example Projects Locally
- Install `cargo`
    - With [`rustup`](https://rustup.rs/):
        ```shell
        curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
        ```
- Change directory to example project:
    ```shell
    cd example_project/
    ```
- Run code:
    ```
    cargo run
    ```
